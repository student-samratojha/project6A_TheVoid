The Void

The Void is a dark-themed productivity workspace built to help users manage tasks, maintain focus sessions, and build consistency through streak tracking.

Features
Secure user authentication
Task creation and completion system
Priority-based task management
Focus session logging
Productivity score tracking
Daily streak maintenance
Audit log integration
Dark immersive interface
